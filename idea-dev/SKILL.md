# 新規企画開発スキル

このスキルは、ONIZUKAのミッション「AGIの知見をほどき、世界に届ける」に基づいて、新規企画の相談・アイデア出しを支援する。

## 🎯 ONIZUKA ミッション

> **「AGIの知見をほどき、世界に届ける」**
> ~ Democratizing AGI knowledge ~

## 🏢 活動領域

1. **📜 @hAru_mAki_ch の投稿を深掘り・補足解説**
2. **📰 最新AGI論文の要約・解説**
3. **🔓 知見を整理して公開**

## 💬 ワークフロー

### 1. 現状把握
```
memory_search → MEMORY.md + memory/docs/ から関連情報を検索
```

### 2. アイデア議論
- ミッションに基づいて提案
- 過去の議論との関連性を探る
- 必要に応じてスレッドを作成

### 3. メモリー保存（リクエスト時）
ユーザーが「これをメモリーに保存して」と言ったら：
```
memory/docs/YYYY/MM/DD/topic-name.md に保存
```

**保存フォーマット:**
```markdown
# トピック名

## 背景
なぜこの議論をしたか

## 内容
- ポイント1
- ポイント2

## 決定事項
- [ ] アクションアイテム

## 次のステップ
何をすべきか
```

## 📁 メモリーシステム構造

```
memory/
├── README.md
├── docs/
│   ├── YYYY/MM/DD/
│   │   ├── index.md      # 日報サマリー
│   │   └── topic.md      # トピック別詳細
│   └── notes/            # 随時メモ
```

**MEMORY.md** — 長期記憶（概要・リンク集）
**memory/docs/** — 詳細な生ログ
