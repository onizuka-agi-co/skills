# Prompt Templates for X Visual

## Default Template

```
A visual illustration of: {text}
Key elements: {keywords}
Style: modern, clean, professional infographic style
Format: suitable for social media sharing
```

## Tech/Programming

```
Technical diagram showing: {text}
Key concepts: {keywords}
Style: clean technical illustration with code-like elements
Colors: dark theme with accent colors
```

## AI/ML Topics

```
Visual explanation of AI concept: {text}
Key terms: {keywords}
Style: modern tech illustration with neural network motifs
Colors: gradient blues and purples
```

## Business/Product

```
Professional infographic about: {text}
Key points: {keywords}
Style: corporate-friendly, clean design
Colors: professional palette
```

## News/Events

```
News visualization of: {text}
Key elements: {keywords}
Style: editorial illustration, attention-grabbing
Colors: bold, high contrast
```

## Customization

Replace placeholders:
- `{text}` - Tweet text (truncated to 100 chars)
- `{keywords}` - Extracted keywords (comma-separated)
- `{hashtags}` - Hashtags from tweet
- `{author}` - Tweet author username

## Tips

1. Keep prompts under 200 words
2. Include style hints for consistent output
3. Specify aspect ratio for platform compatibility
4. Add negative prompts if needed (e.g., "no text, no words")
